### Main functionalities:

- User profile:
    - registration
    - login
    - add donations
    - viewing past donations

- Handling the contact form (send confirmation by e-mail).
- Landing page dynamically displaying:
    - foundations
    - non-governmental organization
    - local donations
    - statistics
